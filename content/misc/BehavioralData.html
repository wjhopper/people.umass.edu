Kahana 2002 Dataset

Description

Data from Kahana, M. J. (2002). Associative symmetry and memory theory. Memory & Cognition, 30(6), 823-840. https://doi.org/10.3758/BF03195769.

Also reported in Rizzuto, D. S., & Kahana, M. J. (2001). An autoassociative neural network model of paired-associate learning. Neural Computation, 13(9), 2075-2092 https://doi.org/10.1162/089976601750399317 .

Format

A data frame with 2160 row and 13 variables. Each variable and its type is described below

subject

[integer] Individual subject identifier

block

[integer] Identifies a group of 24 test trials

test

[integer] Identifies the first or second test of a word pair

order

[integer] Identifies a pairs order of presentation within a round of testing

pair

[integer] Word pair identifier within a given subject and block

xreps

[integer] The number of times a word pair was presented during the study phase

cue_direction

[character] Describes the direction of recall. “forward” indicates that the first word in the pair was presented as cue to recall the second word, and “backward” indicates that the second word in the pair was presented as a cue to recall the first.

same_cue

[character] Indicates whether the cueing direction (forward vs. backward) was consistent between tests. A value of “same” indicates that the same direction was tested on both tests, and a value of “switch” means that the cueing direction reversed between tests.

given

[character] The word given as a recall cue

needed

[character] The word to be recalled in response to the cue word

correct

[logical] Accuracy on the test trial

no_resp

[logical] Indicates whether the participant gave up without giving a response on the test trial. In the raw dataset, these responses were coded as “ANSWER: N”.

time

[integer] The time, in milliseconds, from the onset of the recall cue until a response was given 

# Remove missing observations
Kahana2002 <- filter(Kahana2002, !no_resp)
# add K02 class to enable summary methods
Kahana2002 <- as.K02(Kahana2002)

# Summarize accuracy and RT by repetitions and cue direction

# First, find the mean for each condition for each subject
bySs_xreps_CueDirection <- summary(Kahana2002, subject, xreps, cue_direction)
# Next, average across the means from each subject
# Should reproduce values from Rizutto and Kahana 2001 Table 1
byXreps_CueDirection <- group_by(bySs_xreps_CueDirection, xreps, cue_direction) %>%
  summarise(accuracy = mean(accuracy),
            RT = mean(RT),
            N = n()) %>%
  ungroup()
